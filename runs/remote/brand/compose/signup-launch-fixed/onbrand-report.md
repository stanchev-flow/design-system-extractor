## Method

- **neverDo:** every `neverDo[]` rule in `brand.yaml` is dispatched through a brand-agnostic checker registry (supports both hard-edged editorial id families and rounded-SaaS id families) and statically verified against the rendered HTML/CSS. Unknown ids fall back to a keyword heuristic, else report as informational rather than a false fail.
- **Fidelity:** structured checklist derived from the rendered layout's resolved surface, tokens and componentMapping (no per-brand literals). Each row asserts a token/layout fact from the source crop is actually present in the render. Visual confirmation was done by screenshotting the render with headless Chromium and comparing it beside the source crop in `assets/`.
- **Slop checklist:** generic-AI-look failure modes (default fonts, off-palette hex, lorem copy, missing assets, rounding off the brand radius scale, missing webfonts).
